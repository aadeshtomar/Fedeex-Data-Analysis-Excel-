# FedEx Logistics Data Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing FedEx logistics and shipment data to uncover business insights and improve operational performance. The dashboard provides a comprehensive view of shipment trends, delivery efficiency, revenue generation, shipping costs, and regional performance through interactive visualizations.

## 🎯 Objectives

* Analyze shipment and delivery performance.
* Track key business KPIs.
* Identify trends and patterns in logistics operations.
* Evaluate regional performance and revenue contribution.
* Support data-driven decision-making.

## 🛠️ Tools & Technologies

* Microsoft Excel
* Power BI
* Power Query
* DAX (Data Analysis Expressions)

## 📊 Key Metrics Analyzed

* Total Shipments
* Total Revenue
* On-Time Delivery Rate
* Delivery Status
* Shipping Costs
* Regional Performance
* Customer Distribution
* Transportation Mode Analysis

## 🔍 Project Workflow

1. Data Collection and Understanding
2. Data Cleaning and Preprocessing
3. Data Transformation using Power Query
4. Data Modeling
5. KPI Creation using DAX
6. Dashboard Development
7. Business Insights and Recommendations

## 📈 Dashboard Features

* Interactive Filters and Slicers
* KPI Cards
* Revenue Analysis
* Shipment Trend Analysis
* Delivery Performance Tracking
* Regional Performance Visualization
* Dynamic Reporting

## 💡 Key Insights

* Identified high-performing and low-performing regions.
* Analyzed shipment trends and seasonal patterns.
* Evaluated delivery delays and operational bottlenecks.
* Monitored shipping costs and revenue performance.
* Generated actionable business insights from logistics data.

## 🚀 Business Impact

The dashboard enables stakeholders to monitor logistics performance, optimize delivery operations, reduce inefficiencies, and make informed business decisions using real-time insights.
